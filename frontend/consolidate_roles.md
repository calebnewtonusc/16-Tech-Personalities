# Role Consolidation Analysis

## Current: 95 roles → Target: ~50 roles

### Redundancies Identified

#### Mobile (6 → 2 roles)
**REMOVE:**
- iOS Engineer
- Android Engineer
- Flutter Developer
- React Native Developer
- Cross-Platform Mobile Engineer

**KEEP:**
- Mobile Engineer (covers all platforms)
- Mobile QA Engineer (specialized testing role)

#### Game Development (5 → 1 role)
**REMOVE:**
- Unity Developer
- Unreal Engine Developer
- Gameplay Engineer
- Game Engine Programmer

**KEEP:**
- Game Developer (covers all game dev)

#### Blockchain (3 → 1 role)
**REMOVE:**
- Blockchain Developer
- Smart Contract Engineer

**KEEP:**
- Blockchain Engineer (covers all blockchain work)

#### Security (8 → 4 roles)
**REMOVE:**
- Cloud Security Engineer (merge into Security Engineer)
- Infrastructure Security Engineer (merge into Security Engineer)
- Network Security Engineer (merge into Security Engineer)

**KEEP:**
- Security Engineer (general security)
- Application Security Engineer (AppSec - specialized)
- DevSecOps Engineer (DevSecOps - specialized workflow)
- Penetration Tester (specialized role)
- Security Researcher (specialized research)

#### Infrastructure (9 → 4 roles)
**REMOVE:**
- Container Platform Engineer (merge into Kubernetes Engineer)
- Build Engineer (merge into CI/CD Engineer)
- Release Engineer (merge into CI/CD Engineer)
- Infrastructure Engineer (too similar to Platform Engineer)

**KEEP:**
- Platform Engineer (infrastructure + internal services)
- Site Reliability Engineer (SRE) (specialized reliability)
- DevOps Engineer (CI/CD + operations)
- Kubernetes Engineer (container orchestration specialist)

#### Backend/API (4 → 2 roles)
**REMOVE:**
- API Engineer (merge into Backend Engineer)
- Protocol Engineer (merge into Backend Engineer)

**KEEP:**
- Backend Engineer (covers APIs, microservices, protocols)
- Microservices Engineer (specialized architecture)

#### Machine Learning (7 → 4 roles)
**REMOVE:**
- Applied Machine Learning Engineer (merge into Machine Learning Engineer)
- Generative AI Engineer (merge into LLM Engineer)

**KEEP:**
- Machine Learning Engineer (general ML)
- MLOps Engineer (ML infrastructure)
- LLM Engineer (specialized AI/generative)
- Computer Vision Engineer (specialized domain)
- NLP Engineer (specialized domain)
- Deep Learning Engineer (specialized approach)

#### ML Research (3 → 2 roles)
**REMOVE:**
- Research Engineer (merge into ML Research Engineer)

**KEEP:**
- ML Research Engineer (covers ML research + engineering)
- Research Scientist (pure research, different from engineering)

#### Data Engineering (4 → 2 roles)
**REMOVE:**
- Data Pipeline Engineer (merge into Data Engineer)
- Streaming Data Engineer (merge into Data Engineer)

**KEEP:**
- Data Engineer (covers pipelines, streaming, ETL)
- Analytics Engineer (specialized analytics/BI)

#### Database (6 → 3 roles)
**REMOVE:**
- SQL Developer (merge into Database Engineer)
- NoSQL Engineer (merge into Database Engineer)

**KEEP:**
- Database Engineer (covers SQL/NoSQL)
- Database Administrator (DBA) (specialized operations)
- ETL Developer (specialized data pipelines)
- Data Warehouse Engineer (specialized analytics infrastructure)

#### Testing/QA (5 → 3 roles)
**REMOVE:**
- Software Test Engineer (SDET) (merge into QA Engineer)
- Performance Test Engineer (merge into Performance Engineer)

**KEEP:**
- QA Engineer (general testing + automation)
- Test Automation Engineer (specialized automation)
- Mobile QA Engineer (specialized mobile testing)

#### Performance/Observability (4 → 3 roles)
**REMOVE:**
- Reliability Engineer (merge into Site Reliability Engineer)

**KEEP:**
- Performance Engineer (performance optimization)
- Observability Engineer (monitoring, metrics, traces)
- Site Reliability Engineer (SRE) (reliability engineering)

#### Developer Relations (2 → 1 role)
**REMOVE:**
- Developer Relations Engineer (same as Developer Advocate)

**KEEP:**
- Developer Advocate (covers all DevRel)

#### Developer Tools (3 → 2 roles)
**REMOVE:**
- Developer Experience Engineer (DevEx) (merge into Developer Tools Engineer)

**KEEP:**
- Developer Tools Engineer (covers DevEx, internal tools)
- SDK Engineer (specialized SDK development)

#### Growth/Experimentation (2 → 1 role)
**REMOVE:**
- Experimentation Engineer (merge into Growth Engineer)

**KEEP:**
- Growth Engineer (covers A/B testing, experimentation)

#### Systems (5 → 4 roles)
**REMOVE:**
- Real-Time Systems Engineer (merge into Systems Engineer)

**KEEP:**
- Systems Engineer (distributed systems, real-time)
- Embedded Software Engineer (hardware-focused)
- Firmware Engineer (hardware-focused)
- Compiler Engineer (specialized tooling)

#### Frontend/Web (3 → 2 roles)
**REMOVE:**
- React Native Developer (already covered by Mobile Engineer)

**KEEP:**
- Frontend Engineer (web frontends)
- Web Developer (general web development)
- Web3 Developer (specialized blockchain/web3)

#### AI (3 → 1 role)
**REMOVE:**
- Generative AI Engineer (merge into LLM Engineer)

**KEEP:**
- AI Engineer (general AI engineering)
- LLM Engineer (specialized large language models)
